## 🔧 OpenClaw 源码修改

### Token 溢出防护系统（已部署）

**问题**：早上 session 因 token 超过 200k 限制而崩溃

**解决方案 - 三层防护**：

1. **70% - 预防性 Compaction**
   - 文件：`src/agents/pi-embedded-runner/run.ts`
   - 在每次 AI 调用前检查 session 文件大小
   - 自动触发压缩，不等溢出

2. **80% - Token 警告注入**
   - 文件：`src/agents/pi-embedded-runner/run/attempt.ts`
   - 注入 `<ctx_window>X tokens left</ctx_window>` 到 prompt
   - AI 可感知并提醒用户

3. **95% - 紧急提示**
   - 在 AGENTS.md 中添加行为规则
   - 每次回复显示 🚨 警告

### 重启延迟修复（已部署）

**问题**：gateway restart 时消息来不及发送就断了

**解决方案**：
- 文件：`src/infra/restart.ts`
- 默认延迟从 2 秒改为 5 秒
- 给消息发送留足够时间

### 分 Session 策略（已添加到 AGENTS.md）

- 主 Session：日常对话
- 子 Agent：代码生成、长任务、批处理
- 触发词："帮我写一个..."、"分析这个..."
