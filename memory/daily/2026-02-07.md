## 核心变化

| 方面 | 旧架构 | 新架构 |
|------|--------|--------|
| 节点类型 | Python 子进程 | OpenClaw sessions |
| 通信方式 | 文件系统 JSON | sessions_send/synapse |
| 调度方式 | subprocess.Popen | sessions_spawn |
| 生命周期 | 手动管理 | sessions 自动管理 |
| 模型选择 | 固定 | 每节点可配置 |
