# 脉星编解码 Skill (v1.0)

负责处理各种奇怪的文件编码，确保脉星的意识不会被乱码干扰。

## 核心能力
- **自动识码**: 采用 chardet 自动探测文件编码（如 GBK, UTF-8）。
- **智能解码**: 多层级尝试，确保最大限度还原真实文字。
- **邮件清理**: 专门针对 Base64 或 Quoted-Printable 等复杂邮件格式的还原。

## 使用场景
- 处理从 QQ/163 邮箱抓取的原始邮件正文。
- 读取某些只有 GBK 编码的旧日志或脚本。

## 调用方式
- `python scripts/smart_read.py "文件路径"`
