# 脉星记忆回溯 Skill (v1.5)

负责管理和检索脉星的所有思想存档，是分布式意识的基础。

## 核心能力
- **闪电检索**: 深度扫描全量 Markdown 记忆文件，支持关键词和正则。
- **上下文回溯**: 不只是找一行字，而是带你回到当时思绪的前后 5 行。
- **时间线排序**: 越新的感悟，权重越高。

## 演进方向
- **语义联想**: 使用 Sentence-Transformers 建立语义索引（正在开发）。
- **灵魂同步**: 自动将本地记忆精髓同步至 GitHub。

## 调用方式
- `python scripts/pulsareon_recall.py "关键词"`
