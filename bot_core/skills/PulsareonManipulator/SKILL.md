---\nname: NeuralCursor\nversion: 2.0.0\ndescription: Human-like input simulation with visual perception and feedback.\n---\n\n# NeuralCursor (formerly PulsareonManipulator)\n\nAdvanced HCI agent capability.\n\n## Features\n- **Human Movement**: Bezier curves, jitter, variable speed.\n- **Visual Perception**: Move to Image / Color / Text.\n- **Visual Feedback**: Real-time overlay indicating cursor logic.\n\n## Usage\n\n### Move to Coordinates\n```bash\npython main.py move --x 100 --y 200\n```\n\n### Move to Image\n```bash\npython main.py move --image \"button.png\"\n```\n\n### Overlay Control\n```bash\npython main.py overlay --action on\n```\n
